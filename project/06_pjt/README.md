# 06_pjt - 영화 관리 웹 애플리케이션

> Django와 Bootstrap을 활용하여 영화 관리 기능을 구현하는 프로젝트입니다.  
> 영화 데이터 생성, 조회, 수정, 삭제(CRUD)를 지원하며, 댓글 시스템을 추가하여 사용자 간의 영화에 대한 의견을 나눌 수 있습니다.

---

# 프로젝트 관리 및 협업 방안

## Git 및 GitHub 협업 흐름

### 1. 브랜치 생성 및 관리
- **마스터 브랜치**는 항상 최신 상태로 로컬에서 유지
- 각 팀원은 **기능별 브랜치**에서 작업 후 변경 사항을 **푸시**하여 공유
  - 예시: `feature/create-movie`, `feature/add-comments`
- 커밋 메시지 작성:
  - `feat: 기능 추가`
  - `fix: 버그 수정`
  - `docs: 문서 수정`

### 2. 커밋 메시지 및 기록
- 각 커밋은 **기능 개발**, **버그 수정** 등 주요 작업을 명확히 구분하여 커밋 메시지 작성
  - 예시: `feat: Add movie creation form validation`
- 커밋 후 **GitLab**에 업로드하여 팀원들과 공유

### 3. 브랜치 푸시 및 머지
- **Pull Request**로 코드 리뷰 후 **마스터 브랜치**로 머지
- **리뷰 과정**에서 변경 사항 논의 및 수정

### 4. 협업 중 주의사항
- **자주 최신 상태로 pull**하여 충돌 방지
- **작업 내용 공유**: 커밋 후 상세한 메시지로 팀원과 공유
- **충돌 방지**: 작업을 시작하기 전에 항상 최신 상태로 pull 후 작업

### 5. 프로젝트 종료 시 점검
- **브랜치 관리 원칙**을 README에 기록
- **GitLab**에서 커밋 기록 확인 및 스크린샷으로 기록하기 

### 6. 협업 시 주의사항
- **독립적인 작업** 진행하여 다른 팀원의 작업에 영향 미치지 않기
- **피드백 및 리뷰**를 통해 코드 개선

---

이렇게 정리된 규칙을 따라 협업을 원활하게 진행하고, **버전 관리**와 **문서화**를 통해 프로젝트 관리 효율성을 높입니다.

## ✨ 구현 기능

* **F01: 영화 목록 페이지 (Index)**
  * 전체 영화 목록을 조회하여 화면에 출력
  * 각 영화에 대한 제목, 설명, 그리고 상세 페이지로 가는 링크 제공

* **F02: 영화 생성 페이지 (Create)**
  * 사용자가 영화 데이터를 입력할 수 있는 폼 제공
  * 영화의 제목, 줄거리, 감독, 장르, 평점 등을 입력받아 저장
  * 유효하지 않은 데이터는 저장되지 않으며, 성공적으로 생성된 영화는 상세 페이지로 리다이렉트

* **F03: 영화 상세 페이지 (Detail)**
  * 영화 제목, 줄거리, 감독, 장르, 평점 데이터 표시
  * 댓글 목록과 댓글 작성 기능 제공
  * 영화 수정 및 삭제 기능 제공

* **F04: 영화 수정 페이지 (Update)**
  * 기존 영화 데이터를 수정할 수 있는 폼 제공
  * 수정된 정보는 영화 상세 페이지로 리다이렉트

* **F05: 영화 삭제 (Delete)**
  * 영화 삭제 버튼을 통해 선택한 영화를 삭제
  * 삭제 후 메인 페이지로 리다이렉트
